- firebase

- cypress

# setup

- `npm create svelte@latest`
- `npx svelte-add@latest tailwindcss --tailwindcss-daisyui`
- `npm install`
- `npm run dev -- --open`
- in *tailwind.config.cjs*:
  ```
  daisyui:{
    themes:true,
  }
- in *app.html*: `data-theme="..."`
