# setup

- `npm create svelte@latest`
- `npx svelte-add@latest tailwindcss`
- `tailwindcss-daisyui`
- `npm install`
- `npm run dev -- --open`
- in _tailwind.config.cjs_:
  ```
  daisyui:{
    themes:true,
  }
  ```
- in _app.html_: `data-theme="..."`
